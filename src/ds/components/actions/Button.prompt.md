# Button

Pill button with the signature left-to-right hover sweep. Use for primary and secondary page actions. Renders as `<a>` when `href` is set.

```jsx
<Button variant="primary">Connect</Button>
<Button variant="secondary" href="/approach">Our approach</Button>
<Button variant="secondary" trailingCircle icon="arrow-forward">Meet the team</Button>
```

Variants: `primary` (filled ink), `secondary` (hairline outline), `ghost`. Sizes: `sm` / `md` / `lg`. Copy is uppercase Akkurat Mono. Pair with `IconButton` for circular arrow actions.
