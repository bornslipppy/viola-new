# IconButton

Circular icon button — the round arrow control used throughout the site (nav, card links, back-to-top). Fills with the ink sweep on hover.

```jsx
<IconButton icon="arrow-forward" label="Next" />
<IconButton icon="arrow-outward" variant="filled" label="Open" href="/x" />
```

`variant`: `outline` (default) / `filled`. `size`: `sm` (36px) / `md` (45px). Always pass `label`.
