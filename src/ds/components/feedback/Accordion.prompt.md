# Accordion

Bordered accordion rows with Season Serif triggers and a `+` glyph that rotates to `×` when open. Used for FAQ / disclosure sections.

```jsx
<Accordion items={[
  { title: "When do you invest?", content: "After the first spark…" },
  { title: "How do you partner?", content: "Hands-on, day in and day out." },
]} />
```
